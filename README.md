Boxus
=====

## About
Open automated grow pod project


## Requirements

### Software

* Nanpy https://github.com/nanpy/
* Adafruit Python DHT https://github.com/adafruit/Adafruit_Python_DHT

### Hardware

* Raspberry Pi 3
* Arduino Nano v3
* DHT temperature and humidity sensor (DHT 11 tested)
* Soil moisture sensor
